# 将JSON转换为JS对象

js中有一个工具类，叫JSON

这个对象可以将一个JSON对象转换为JS对象

也可以将一个JS对象转换为JSON

 

JSON.parse()

可以将JSON字符串转换为js对象

要一个JSON字符串作为参数

 

JSON.stringify()

将JS对象转换为JSON字符串

要一个js对象作为参数，返回一个JSON字符串
