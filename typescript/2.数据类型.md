# Array

### 第一种定义数组的方式

指定数组包含类型，非此类型会报错

```typescript
let arr:number[] = [1,2,3]
```

### 第二种方式

使用==泛型==来定义

```typescript
let arr:Array<number> = [1,2,3]
```

# tuple

元组类型，属于数组的一种，可以指定数组里面值的类型

```ts
let tuple:[string,number] = ['tom',12]
```
