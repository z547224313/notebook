# 使用

### 创建项目

使用npx 不用全局安装 create-react-app也可以使用该包

```shell
npx create-react-app 项目名称
```

### PWA

渐进式网页开发技术

可以将网页离线安装

public目录下的manifest.json就是干这个的

Web App 功能

- 可以在手机上将网页添加到主屏幕
- 实现离线缓存，没网可以使用离线功能
- 实现消息推送

### yarn

eject 暴露配置文件

### 入口

index.js

```js
import ReactDOM from 'react-dom'
import React from 'react'
ReactDOM.render(<h2>Hello</h2>,document.getElementById('root'))

```
