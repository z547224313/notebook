# 起步

创建一个package.json文件

`npm init`

在该目录下安装express

`npm install express --save`

# HelloWorld

```js
const express = require('express')

const app = express()

app.get('/',(req,res)=>{
  res.send('Hello World')
})

app.listen('3000',()=>{
  console.log('3000端口服务器已启动');
  
})
```

1. 导包
2. app = express() 类似 serve = http.createServer()
3. app.listen() 开启端口服务
4. get监听

### 开启静态资源库API

```js
// 开启public静态资源文件
app.use('/public/',express.static('./public/'))
```
