# 爬楼梯

10层楼梯，每次上1或2 一共几种方式

到10之前的状态 9 8

假定9 8 次数为 X Y

则10 为 X+Y则有 fn = fn-1+fn-2

且有 f(0) = 0

f(1)= 1

f2=2

### 递归

```js
function fn(n){
  if(n==1){
    return 1
  }
  if(n==2){
    return 2
  }
  if(n==0){
    return 0
  }
  return fn(n-1)+fn(n-2)
}
```

### 动态规划

```js
function fn(n){
  if(n==1){
    return 1
  }
  if(n==2){
    return 2
  }
  if(n==0){
    return 0
  }
  if(n>=3){
    let a= 1
    let b=2
    let temp = 0
    for(let i =3 ;i<=n;i++){
      temp = a+b
      a=b
      b=temp
    }
    return temp
  }
}
```
