```html
<!DOCTYPE html>
<!-- <!DOCTYPE html> 声明是一个html5页面-->
<html lang='en'>
<!-- lang = 'en' 声明 -->
<head>
  <meta charset="utf-8">
  <!-- meta标签 设置一些元数据，元数据给搜索引擎看 -->
  <!--
      name 指定数据的名称
      content 指定数据的内容


      keywords 表示网站的关键字


      http-equiv="refresh" content="3" 将页面三秒后重定向到其他页面
   -->
  <meta name='keywords' content="HTML5,前端">
  <meta name='description' content="这是描述">
  <meta http-equiv="refresh" content="3;url=https://www.baidu.com">
  <title>html结构说明</title>
</head>
</html>
```

meta 元数据给浏览器看
